Day 39 Short Hand If Else


Short-hand if-else(ternary operator):
In Python, when you want to write a simple if-else statement that only returns or assigns a value based on a condition, you can use a short-hand or one-line format.
This makes your code more concise and readable, especially for simple decisions.

Syntax of Short-hand if-else:
value_if_true if condition else value_if_false.

But For complex logic or when you have multiple lines to run under each condition, stick with the full if-else block, not the shorthand
