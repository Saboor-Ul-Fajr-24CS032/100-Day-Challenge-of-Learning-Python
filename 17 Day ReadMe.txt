Day 17 -> Introductions to Lists


List in Python?
A list in Python is a collection of items (elements) that is:
Ordered → Items have a fixed position.
Mutable → You can change items after creating the list.
Allows duplicates → Can store repeated items.
Can store different data types in one list.
List items are enclosed in square brackets [ ] and separated by commas.

List Indexing
Each item in a list has an index (position number), starting from 0.
