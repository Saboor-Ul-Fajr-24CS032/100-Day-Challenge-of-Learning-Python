Day 63 -> Time Module in Python


Time module:
The time module allows Python programs to handle time-related tasks like getting the current time, pausing the program, or formatting dates/times.

Three functions that i use 

1. time.time():
Returns current time in seconds (as float) since the epoch (January 1, 1970).

2. time.sleep(seconds):
Pauses program for the given number of seconds.

3. time.strftime(format, time_object):
Formats time into a readable string.
