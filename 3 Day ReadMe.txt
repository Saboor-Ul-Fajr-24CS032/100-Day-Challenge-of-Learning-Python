Day 3 -> Comments and scape sequence

Comments in Python:

. Comments are notes in the code that Python ignores; they help explain what the code does.

. Single-line comment: Starts with #.

. Multi-line comment: Use triple quotes ''' or """.

Escape Sequences in Python:

. Escape sequences are special codes in strings that do something instead of showing as text.

. They Starts with a backslash \.

. Common types: \n (new line), \t (tab), \\ (backslash), \" (double quote).
