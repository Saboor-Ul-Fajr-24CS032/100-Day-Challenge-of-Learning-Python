Day 27 -> Dictionaries method 

Update:
The .update() function adds new key-value pairs or changes the value of existing keys in the dictionary.

Clear:
The .clear() function removes all items from the dictionary, leaving it empty.

Pop:
pop() function removes the item with the specified key and returns its value without that key and it's value.

Popitem:
The .popitem() function removes and returns the last key-value pair added to the dictionary.

Del:
The del keyword deletes a specific key-value pair or the whole dictionary from memory.
