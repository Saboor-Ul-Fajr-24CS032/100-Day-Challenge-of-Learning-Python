Day 18 -> Lists Methods



1. append()
Use: Adds a single element to the end of the list.
Syntax: list.append(element)

2. count()
Use: Counts how many times a specific value appears in the list.
Syntax: list.count(value)

3. copy()
Use: Returns a shallow copy of the list (a new list with the same elements).
Syntax: list.copy()

4. insert()
Use: Inserts an element at a specific index.
Syntax: list.insert(index, value)

5. extend()
Use: Adds elements from another iterable (like list or tuple) to the end of the current list.
Syntax: list.extend(iterable)

6. Concatenation (+)
Use: Joins two lists into one.
Syntax: list1 + list2

7. sort()
Use: Sorts the list in ascending order by default.
Syntax: list.sort() or list.sort(reverse=True) for descending.

8. reverse()
Use: Reverses the order of elements in the list.
Syntax: list.reverse()

9. index()
Use: Returns the first index of the given value.
Syntax: list.index(value)
