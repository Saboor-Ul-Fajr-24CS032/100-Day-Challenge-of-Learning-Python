Day 16 -> Function Arguments 


Function Arguments:
When you define a function, you can pass data to it using arguments.
Arguments are values you provide to a function so it can perform its task using that data.


Types of Function Arguments:
In Python, there are four main types of arguments:

Positional Arguments
They are the most common type.
Arguments are matched to parameters by their position/order.

Keyword Arguments
You specify the parameter names when calling the function.
It's order doesn’t matter.

Default Arguments:
You provide default values in the function definition.
If no value is passed, the default is used.

Variable-length Arguments:
Sometimes you don’t know how many arguments will be passed.
You can handle this with:
*variable_name -> For multiple positional arguments


Return Statement:
The return statement is used in a function to send data back to the caller.
It ends the function.
The function outputs a value, which can be used later.
